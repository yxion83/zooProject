# Zoo Application

## Overview
This is a student project built in C# using Console and WPF as part of my Object-Oriented Programming class at Northcentral Technical College (NTC).  
The app allows users to explore zoo exhibits, view animal details, etc and interact with a simple, user-friendly interface.  

The project helped me practice and apply programming concepts such as classes, inheritance, collections, and UI development in WPF.

## Features
- Browse different zoo exhibits
- View information about each animal
- Interactive interface using WPF
- Console-based functionality for quick testing

## Technologies Used
- C#
- WPF 
- Console applications
- Object-Oriented Programming principles
